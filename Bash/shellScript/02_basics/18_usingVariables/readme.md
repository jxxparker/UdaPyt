2.18 : Variables

    - We have 3 ways how to assign a value to a variable.
        - explicit definition : VAR=value
        - Read command: read VAR
        - Command substitution: VAR=$(pwd)

    1 : Explicit Definition
        - VAR=value
        - PATH=/var/lib
        - DOGS_NUMBER=8
        - MESSAGE=hello
        - MESSAGE=hello buddy
        - MESSAGE="hello buddy"

    2 : Access Variable
        - echo $COUNT
        - echo $PATH
