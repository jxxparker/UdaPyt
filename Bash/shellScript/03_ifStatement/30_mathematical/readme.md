3.30 : Mathematical Comparison

- Operands
    : -eq : equal to
    : -ne : not equal to
    : -gt : greater than
    : -lt : less than
    : -ge : greater than or equal to
    : -le : less than or equal to

- Examples :
- if [$VAR -eq 5]; then
- if [$VAR -gt 7]; then
- if [$VAR -eq 5] && [$COUNT -ne 1]; then
